# 🚀 Nodami - 自建梯子节点，从未如此简单

**Nodami** 是一款开源代理服务端管理平台，专为简化自建梯子节点而设计。它可以轻松实现自动部署、集中管理多个服务器节点，完美兼容 Shadowsocks / Vmess / Hysteria 等多种协议，提供稳定、安全且高性能的网络连接。

---

## 🌟 项目特性

- **极简部署**：无需专业知识，只需简单填写服务器账号密码即可自动搭建梯子服务。
- **多服务器集中管理**：轻松管理多个服务器节点，统一界面清晰直观。
- **实时状态监控**：提供服务器节点状态实时监听，轻松掌控节点健康状况。
- **多协议支持**：兼容多种常用代理协议，包括 Shadowsocks、Vmess 和 Hysteria 等。
- **高性能 & 安全可靠**：精心优化的连接管理和安全机制，保障稳定高速的连接体验。

---

## 🛠️ 安装教程

### 环境准备
- Linux 服务器（推荐 Ubuntu / CentOS）

### 快速安装 一键安装
```bash
curl -fsSL https://raw.githubusercontent.com/YoyoCrafts/Nodami/refs/heads/nodami/docker/install.sh -o install.sh && sudo bash install.sh
```
